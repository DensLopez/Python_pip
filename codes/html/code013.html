Pandas¶

Es una librería de Python que nos sirve para manipular datos provenientes de diferentes formatos como CSV, XLS, entre otros.

Regresemos al proyecto app, y procedamos a activar el entorno.

source env/bin/activate
which python3
#instalar pandas
pip3 install pandas
#verificar librerías
pip3 freeze
#actualizar requirements.txt
pip3 freeze > requirements.txt

Vamos a actualizar el main.py, para poder usar pandas.

Una vez que hemos modificado el archivo main.py procederemos a ejecutarlo en la terminal.
